## üéØ Overview

This project provides a comprehensive, interactive guide to ECMAScript (JavaScript's official specification) covering all versions from ES1 (1997) to the latest ES2024. It's designed as an educational resource for developers of all skill levels, from beginners to advanced practitioners.

### What is ECMAScript?

ECMAScript is the official specification that defines how JavaScript should work. It's maintained by ECMA International and ensures consistent behavior across all browsers and JavaScript engines. Think of it as the "rulebook" that all JavaScript implementations must follow.

### Key Benefits

- **Cross-Platform Consistency**: Ensures JavaScript works the same way across all browsers
- **Standardized Evolution**: Provides a clear roadmap for JavaScript's future
- **Developer Confidence**: Allows developers to write code that works reliably everywhere
- **Annual Releases**: Since ES2015, new features are released annually

## üìÖ ECMAScript Versions

### Early Years (1997-2009)

#### ES1 (1997) - First Edition
- **Release**: June 1997
- **Key Features**:
  - Basic syntax and grammar
  - Variables and data types
  - Control structures (if, while, for)
  - Functions
  - Objects and arrays

#### ES2 (1998) - Second Edition
- **Release**: June 1998
- **Key Features**:
  - Minor updates to align with ISO/IEC 16262
  - International standardization

#### ES3 (1999) - Third Edition
- **Release**: December 1999
- **Key Features**:
  - Regular expressions
  - try-catch exception handling
  - Better string methods
  - Formatting for numeric output
  - Error handling improvements

#### ES5 (2009) - Fifth Edition
- **Release**: December 2009
- **Key Features**:
  - Strict mode (`'use strict'`)
  - JSON support
  - Array methods: `map()`, `filter()`, `reduce()`, `forEach()`
  - Object getters and setters
  - Property descriptors
  - `Object.create()`

### Modern Era (2015-Present)

#### ES6/ES2015 - Sixth Edition
- **Release**: June 2015
- **Key Features**:
  - `let` and `const` declarations
  - Arrow functions
  - Classes
  - Template literals
  - Destructuring assignment
  - Modules (import/export)
  - Promises
  - Symbols
  - Generators
  - Default parameters
  - Rest/spread operators
  - Computed property names

#### ES7/ES2016
- **Release**: June 2016
- **Key Features**:
  - `Array.prototype.includes()`
  - Exponentiation operator (`**`)

#### ES8/ES2017
- **Release**: June 2017
- **Key Features**:
  - `async/await`
  - `Object.values()` and `Object.entries()`
  - String padding (`padStart`, `padEnd`)
  - `SharedArrayBuffer` and `Atomics`

#### ES9/ES2018
- **Release**: June 2018
- **Key Features**:
  - Rest/spread properties for objects
  - `Promise.prototype.finally()`
  - Asynchronous iteration
  - RegExp improvements

#### ES10/ES2019
- **Release**: June 2019
- **Key Features**:
  - `Array.prototype.flat()` and `flatMap()`
  - `Object.fromEntries()`
  - `String.prototype.trimStart()` and `trimEnd()`
  - Optional catch binding

#### ES11/ES2020
- **Release**: June 2020
- **Key Features**:
  - Optional chaining (`?.`)
  - Nullish coalescing (`??`)
  - `BigInt`
  - `Promise.allSettled()`
  - Dynamic `import()`

#### ES12/ES2021
- **Release**: June 2021
- **Key Features**:
  - `String.prototype.replaceAll()`
  - `Promise.any()`
  - Logical assignment operators
  - Numeric separators

#### ES13/ES2022
- **Release**: June 2022
- **Key Features**:
  - Class fields (public and private)
  - Top-level `await`
  - `Array.prototype.at()`
  - `Object.hasOwn()`
  - Error cause

#### ES14/ES2023
- **Release**: June 2023
- **Key Features**:
  - `Array.prototype.findLast()` and `findLastIndex()`
  - Hashbang syntax
  - Symbol as WeakMap keys

#### ES15/ES2024
- **Release**: June 2024
- **Key Features**:
  - `Promise.withResolvers()`
  - Array grouping methods
  - Set methods
  - Pipeline operator (proposal)

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ecmascript-guide.git
   cd ecmascript-guide
   ```

2. Open `index.html` in your browser or serve it using a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

3. Navigate to `http://localhost:8000` in your browser
