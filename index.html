What is ECMAScript?

Standard Specification

ECMAScript is the official specification that defines how JavaScript should work. It's maintained by ECMA International and ensures consistent behavior across all browsers and JavaScript engines.

Cross-Platform

By following the ECMAScript standard, developers can write JavaScript code that works consistently across Chrome, Firefox, Safari, Edge, and other browsers.

Annual Releases

Since ES2015, ECMAScript follows an annual release cycle, introducing new features and improvements to make JavaScript more powerful and developer-friendly.

ECMAScript Versions Timeline

1997

ES1 - First Edition

The original ECMAScript standard that established the foundation of JavaScript.

  • Basic syntax and grammar
  • Variables and data types
  • Control structures
  • Functions
1998

ES2 - Second Edition

Minor updates to align with ISO/IEC 16262 international standard.

1999

ES3 - Third Edition

Significant improvements including regular expressions, better string handling, and exception handling.

  • Regular expressions
  • try-catch exception handling
  • Better string methods
  • Formatting for numeric output
2009

ES5 - Fifth Edition

A major update that introduced strict mode and many new methods for arrays and objects.

  • Strict mode ('use strict')
  • JSON support
  • Array methods: map, filter, reduce
  • Object getters and setters
  • Property descriptors

ES5 Example:

'use strict';

var numbers = [1, 2, 3, 4, 5];
var doubled = numbers.map(function(n) {
    return n * 2;
});

console.log(doubled); // [2, 4, 6, 8, 10]
2015

ES6/ES2015 - Sixth Edition

The most significant update in ECMAScript history, introducing modern JavaScript features.

  • let and const declarations
  • Arrow functions
  • Classes
  • Template literals
  • Destructuring
  • Modules (import/export)
  • Promises
  • Symbols
  • Generators

ES6 Example:

// Arrow functions and template literals
const greet = (name) => `Hello, ${name}!`;

// Destructuring
const { title, author } = book;

// Classes
class Person {
    constructor(name) {
        this.name = name;
    }
    
    greet() {
        return greet(this.name);
    }
}
2016

ES7/ES2016

Small but useful additions to the language.

  • Array.prototype.includes()
  • Exponentiation operator (**)

ES7 Example:

// Exponentiation operator
const result = 2 ** 3; // 8

// Array includes
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.includes('banana')); // true
2017

ES8/ES2017

Introduced async/await for better asynchronous programming.

  • async/await
  • Object.values() and Object.entries()
  • String padding (padStart, padEnd)
  • SharedArrayBuffer and Atomics

ES8 Example:

// Async/await
async function fetchUser(id) {
    try {
        const response = await fetch(`/api/users/${id}`);
        const user = await response.json();
        return user;
    } catch (error) {
        console.error('Error:', error);
    }
}

// Object methods
const obj = { a: 1, b: 2, c: 3 };
console.log(Object.values(obj)); // [1, 2, 3]
console.log(Object.entries(obj)); // [['a', 1], ['b', 2], ['c', 3]]
2018

ES9/ES2018

Enhanced object and promise features.

  • Rest/spread properties for objects
  • Promise.prototype.finally()
  • Asynchronous iteration
  • RegExp improvements

ES9 Example:

// Object rest/spread
const { x, y, ...rest } = { x: 1, y: 2, a: 3, b: 4 };
console.log(rest); // { a: 3, b: 4 }

const obj = { ...rest, z: 5 };

// Promise.finally
fetch('/api/data')
    .then(response => response.json())
    .catch(error => console.error(error))
    .finally(() => console.log('Request completed'));
2019

ES10/ES2019

Array and string improvements.

  • Array.prototype.flat() and flatMap()
  • Object.fromEntries()
  • String.prototype.trimStart() and trimEnd()
  • Optional catch binding

ES10 Example:

// Array flat
const nested = [1, [2, 3], [4, [5, 6]]];
console.log(nested.flat()); // [1, 2, 3, 4, [5, 6]]
console.log(nested.flat(2)); // [1, 2, 3, 4, 5, 6]

// Object.fromEntries
const entries = [['name', 'John'], ['age', 30]];
const obj = Object.fromEntries(entries);
console.log(obj); // { name: 'John', age: 30 }
2020

ES11/ES2020

Null safety and module improvements.

  • Optional chaining (?.)
  • Nullish coalescing (??)
  • BigInt
  • Promise.allSettled()
  • Dynamic import()

ES11 Example:

// Optional chaining
const user = { profile: { name: 'John' } };
console.log(user?.profile?.name); // 'John'
console.log(user?.address?.street); // undefined

// Nullish coalescing
const value = null ?? 'default'; // 'default'
const empty = '' ?? 'default'; // ''

// BigInt
const bigNumber = 9007199254740991n;
console.log(bigNumber + 1n); // 9007199254740992n
2021

ES12/ES2021

String and logical improvements.

  • String.prototype.replaceAll()
  • Promise.any()
  • Logical assignment operators
  • Numeric separators

ES12 Example:

// String replaceAll
const text = 'hello world hello';
console.log(text.replaceAll('hello', 'hi')); // 'hi world hi'

// Logical assignment
let x = 0;
x ||= 5; // x = 5 (if x is falsy)
x &&= 10; // x = 10 (if x is truthy)

// Numeric separators
const million = 1_000_000;
console.log(million); // 1000000
2022

ES13/ES2022

Class fields and top-level await.

  • Class fields (public and private)
  • Top-level await
  • Array.prototype.at()
  • Object.hasOwn()
  • Error cause

ES13 Example:

// Class fields
class User {
    name = 'John'; // public field
    #password = 'secret'; // private field
    
    constructor(age) {
        this.age = age;
    }
}

// Top-level await
const response = await fetch('/api/data');
const data = await response.json();

// Array.at()
const arr = [1, 2, 3, 4, 5];
console.log(arr.at(-1)); // 5 (last element)
console.log(arr.at(-2)); // 4 (second to last)
2023

ES14/ES2023

Array and object improvements.

  • Array.prototype.findLast() and findLastIndex()
  • Hashbang syntax
  • Symbol as WeakMap keys

ES14 Example:

// Array findLast
const numbers = [1, 2, 3, 4, 5, 6];
const lastEven = numbers.findLast(n => n % 2 === 0);
console.log(lastEven); // 6

const lastEvenIndex = numbers.findLastIndex(n => n % 2 === 0);
console.log(lastEvenIndex); // 5

// Hashbang
#!/usr/bin/env node
console.log('This is a Node.js script');
2024

ES15/ES2024

Latest features and improvements.

  • Promise.withResolvers()
  • Array grouping methods
  • Set methods
  • Pipeline operator (proposal)

ES15 Example:

// Promise.withResolvers
const { promise, resolve, reject } = Promise.withResolvers();

// Array grouping
const items = [
    { name: 'apple', category: 'fruit' },
    { name: 'carrot', category: 'vegetable' },
    { name: 'banana', category: 'fruit' }
];

const grouped = Object.groupBy(items, item => item.category);
console.log(grouped);
// { fruit: [...], vegetable: [...] }

Interactive Feature Explorer

Variable Declarations

var (ES1)

Function-scoped variable declaration

var x = 10;
if (true) {
    var x = 20; // Same variable
}
console.log(x); // 20

let (ES6)

Block-scoped variable declaration

let x = 10;
if (true) {
    let x = 20; // Different variable
}
console.log(x); // 10

const (ES6)

Block-scoped constant declaration

const PI = 3.14159;
// PI = 3.14; // Error!

const obj = { name: 'John' };
obj.name = 'Jane'; // OK - object properties can change

Function Declarations

Function Declaration (ES1)

function greet(name) {
    return 'Hello, ' + name;
}

Function Expression (ES3)

const greet = function(name) {
    return 'Hello, ' + name;
};

Arrow Function (ES6)

const greet = (name) => {
    return 'Hello, ' + name;
};

// Shorthand
const greet = name => 'Hello, ' + name;

Object Features

Object Literal (ES1)

const person = {
    name: 'John',
    age: 30,
    greet: function() {
        return 'Hello, ' + this.name;
    }
};

Method Shorthand (ES6)

const person = {
    name: 'John',
    age: 30,
    greet() {
        return 'Hello, ' + this.name;
    }
};

Destructuring (ES6)

const { name, age } = person;
console.log(name, age); // John 30

// With default values
const { name, age, city = 'NYC' } = person;

Array Methods

forEach (ES5)

const numbers = [1, 2, 3];
numbers.forEach(n => console.log(n));

map (ES5)

const doubled = numbers.map(n => n * 2);
console.log(doubled); // [2, 4, 6]

filter (ES5)

const evens = numbers.filter(n => n % 2 === 0);
console.log(evens); // [2]

Asynchronous Programming

Callbacks (ES3)

function fetchData(callback) {
    setTimeout(() => {
        callback('Data received');
    }, 1000);
}

Promises (ES6)

function fetchData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve('Data received');
        }, 1000);
    });
}

async/await (ES8)

async function getData() {
    try {
        const data = await fetchData();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}

Interactive Code Examples

Template Literals (ES6)

template-literals.js 
const name = 'John';
const age = 30;
const city = 'New York';

// Old way (ES5)
const message1 = 'Hello, my name is ' + name + 
                ', I am ' + age + ' years old, ' +
                'and I live in ' + city + '.';

// New way (ES6)
const message2 = `Hello, my name is ${name}, 
I am ${age} years old, and I live in ${city}.`;

console.log(message1);
console.log(message2);

Destructuring (ES6)

destructuring.js 
// Object destructuring
const user = {
    name: 'Alice',
    age: 25,
    email: 'alice@example.com',
    address: {
        city: 'Boston',
        country: 'USA'
    }
};

const { name, age, address: { city } } = user;

console.log(name); // Alice
console.log(age);  // 25
console.log(city); // Boston

// Array destructuring
const colors = ['red', 'green', 'blue'];
const [first, second, third] = colors;

console.log(first, second, third);

Optional Chaining (ES2020)

optional-chaining.js 
const user = {
    name: 'John',
    profile: {
        avatar: {
            url: 'https://example.com/avatar.jpg'
        }
    }
};

// Without optional chaining
const avatarUrl1 = user.profile && 
                  user.profile.avatar && 
                  user.profile.avatar.url;

// With optional chaining
const avatarUrl2 = user.profile?.avatar?.url;

console.log('Old way:', avatarUrl1);
console.log('New way:', avatarUrl2);

// With nullish coalescing
const defaultUrl = user.profile?.avatar?.url ?? 'default-avatar.jpg';
console.log('Default:', defaultUrl);
